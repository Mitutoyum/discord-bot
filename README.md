# A simple open source discord bot written in discord.py

### Installation
- Install dependencies
```
pip install -r requirements.txt
```
- Bot setup
```
python main.py setup
```
- Run the bot
```
python main.py run
```

<details>
  <summary><h3>Dependencies</h3></summary>

- `discord.py`
- `aiosqlite`
- `typer`
- `python-dotenv`
</details>
